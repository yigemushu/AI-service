# 浏览器插件 MVP

这个插件用于验证电脑端流程：在网页聊天页面选中客户消息，然后发送到网站 `/api/inbox`，再回到消息中心同步。

## 安装方式

1. 打开 Chrome 或 Edge。
2. 进入扩展管理页面：
   - Chrome：`chrome://extensions`
   - Edge：`edge://extensions`
3. 打开“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择本文件夹：

```text
integrations/browser-extension
```

## 使用方式

1. 打开闲鱼网页版或任意客户聊天网页。
2. 用鼠标选中客户发来的消息。
3. 点击浏览器右上角插件图标。
4. 确认网站地址，例如：

```text
http://localhost:3000
```

线上时改成你的正式网站地址。

5. 选择平台和业务类型。
6. 点击“读取网页选中文本”。
7. 点击“发送到消息中心”。
8. 回到网站 `/messages`，点击“同步外部消息”。

## Token

如果服务器设置了：

```env
INBOX_WEBHOOK_TOKEN=
```

插件里也要填写同一个 token。不要把真实 token 提交到 GitHub。

## 当前限制

- 这是验证版，不会自动发送消息。
- 需要用户自己选中客户消息。
- 只负责把消息送进网站，AI 分析和回复确认仍在网站内完成。
- 后续可以升级为自动识别聊天气泡和侧边栏建议回复。
